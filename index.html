About me

I completed my bachelor's at the University of Melbourne, majoring in Computer Science. I am very interested in Game Development, Tools Development, Technical Art and Artificial Intelligence. I have found my passion in Software Engineering but I have varying levels of experience in different fields ranging from Graphic Design to Biology. As a very curious person, I enjoy learning new things more than anything else. The programming languages that I am most familiar with are Python, C#, C++, Java, C and Javascript among others. Portfolio

  • Multiplayer 3rd Person Shooter

    Unreal Engine 5

    I wanted to really dive deeper into Unreal Engine 5 and using C++ to develop games. For this project I got a lot of help from several Udemy courses, and used a lot of free assets available online. Players can host and join sessions using Steam for a Death Match that lasts 10 minutes. Contact me if you want to try it out!

  • FPS Game

    Unreal Engine 5

    I wanted to finish a short project using UE5 just to get used to using it. Finished this project as part of a short online tutorial, and the project is made entirely with blueprints and free assets. This was an easy little project but turned out looking good so I wanted to share it.

  • Chess Set

    Maya

    In an effort to learn the basics of Maya I completed this chess set. In the process I ended up learning some basic modeling, texturing, subsurface scattering, lights, rendering with Arnold and a bit of animation. Click the image to see a 5 second animation showing off the model.

  • Procedural Buildings

    Houdini

    Second attempt at making procedural buildings. Instead of having the shape be completely random, it receives an input curve from the user and generates the building from that. Thinking of using these in an XCOM style game in Unity. I will probably add fire escapes and other cool stuff at some point.

  • Procedural Medieval House

    Houdini

    This is my first ever Houdini project. In order to learn Houdini I decided to try a procedural house generator. The user can play around with number of floors, interesting wall patterns, randomizations for window and door types and locations, balconies, interesting roof patterns and more!

  • Go 13

    pico-8

    I have been obsessed with GO recently, so I wanted to create my own 2 player Go game for Pico-8. Its not a full size 19x19 game. Instead its a quick 13x13. The project was on the simpler side, thanks to the Tromp-Taylor Rules I found online which describe the logical rules of GO.

  • Platform Puzzle

    Unreal Engine 4

    A weekend project I made using Unreal Engine 5. For this game I only used Blueprint Visual Scripting to learn quick prototyping using Unreal. The game is a simple 3D platformer where you collect gems to complete levels, and there are only 4 levels.

  • Lava Jump

    pico-8

    For this project I wanted to try a 3D game with Pico-8. Since Pico-8 doesn't have native 3D support, I followed a few tutorials to implement a 3D world with ray casting. You jump using the O button ("Z" on desktop by default). I recommend enabling autorun from the menu if playing on mobile.

  • Gin Rummy

    pico-8

    One of my favourite card games, Gin Rummy, developed on Pico-8. The hardest part of this project was writing the algorithm to find the ideal card arrangements and calculating deadwood cards. Use the "X" key to select a card, and the arrow buttons to move the selected card. There is a how to play section under options ("P" on desktop).

  • Half Pipe

    pico-8

    A reaction game I developed in the course of a weekend, in order to learn PICO-8. You simply have to press the correct button combinations. (X -> "X" key on desktop, O -> "Z" key on desktop)

  • The Adventure of Mr. Boaty McBoatFace

    Unity

    A 3D Unity game a few friends and I made for our Graphics and Interaction course at the University of Melbourne. Our course was focused on writing shaders but we also tried to make a fun game out of it.

  • Peg Break

    Java

    The first game I made for a course I was taking at the University of Melbourne. Made using Java without the use of an engine. It is a simple game where you click on the screen to shoot a ball downwards to break as many pegs as possible.
