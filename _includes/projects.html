Peg Break

The first game I made for a course I was taking in the University of Melbourne
Made using Java, it is a simple game where you click on the screen to shoot a ball downwards. The ball is affected by gravity, breaks certain pegs, collects power-ups.

Half Pipe

A reaction game I developed in the course of a weekend, in order to learn PICO-8. Click on the image to try the game!
You simply have to press the correct button combinations. You can press and hold the direction before pressing the action key.
(X -> "X" key on the keyboard. O -> "Z" key on the keyboard)
(PS: Music: CKY - 96 Quite Bitter Beings)






Gin Rummy

The popular card game Gin Rummy, developed on Pico-8
The hardest part of this project was writing the algorithm to find the ideal card arrangements and calculating deadwood cards
You can try the game by clicking the image on the left. Use the "X" key to select a card, and the arrow buttons to move the selected card.
There is a how to play section under options ("P" or "Enter" if you are playing on a computer)






Lava Jump

For this project I wanted to try a 3D game with Pico-8. Click on the image to try it!
Since Pico-8 doesn't have native 3D support, I followed a few tutorials to implement a 3D world with ray casting.
The game is a simple 3D platformer, you jump using the O button ("Z" on the keyboard by default), and avoid falling in Lava
My goal was to have something playable in a week. I might come back to it in the future to improve the game. (PS: Music: Deftones - My Own Summer)
